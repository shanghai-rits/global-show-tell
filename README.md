
# Global Show & Tell

## Local Development

1. Navigate to the project folder:
   ```bash
   cd web-ui
   ```
2. Install dependencies:
   ```bash
   nvm use
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:5174/ in your browser to see the app running.

## Deployment

Deployment is automatic via GitHub Actions. Simply:

1. Commit your changes:
   ```bash
   git add .
   git commit -m "your commit message"
   ```
2. Push to the repository:
   ```bash
   git push
   ```

That's it! The site will be automatically built and deployed.

- **GitHub Actions:** https://github.com/shanghai-rits/global-show-tell/actions
- **Live site:** https://nyuglobal.show/
